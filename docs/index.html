Chapter 1

ZX Spectrum 16K

Sinclair Research Ltd released the ZX Spectrum home computer in the UK, in April 1982. There were two memory configurations: 16 KB of RAM (£125), and 48 KB (£175).

Sinclair ZX Spectrum 16K / Hungry Horace (ROM version) by Psion & Melbourne House

Some early Spectrum games were available on ROM cartridge, though not many were released this way, because they were expensive and required an additional peripheral, the Interface 2. In Hungry Horace from 1982, you control a… surrealistic nightmare trying to escape from angry, puce disembodied heads. 

// JSSpeccy configuration
emu = JSSpeccy(
    document.getElementById('jsspeccy'),
    {
        machine: 16,
        roms: { Spectrum48K: './G13R_ROM.bin' }
        uiEnabled: false,
        zoom: 2,
    })

In the configuration above, we’ve selected the 16 KB Spectrum machine (“machine: 16”), and specified the URL path to an uploaded Hungry Horrace ROM ('./G13R_ROM.bin'). (The ROM is specified with “Spectrum48K:”, because the 16 KB and 48 KB models share essentially the same hardware.)

For more detail see the JSSpeccy manual.
